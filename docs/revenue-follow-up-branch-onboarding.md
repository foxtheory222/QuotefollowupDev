# Branch Onboarding

Status: Documented and ready for Admin workflow.

Use this process for each branch before production rollout:

1. Create or verify the branch row.
2. Configure source feeds.
3. Add Manager membership.
4. Add GM membership.
5. Add TSR staff memberships.
6. Add CSSR staff memberships.
7. Map AM Number aliases to TSR staff.
8. Map CSSR Number aliases to CSSR staff.
9. Set active branch policy.
10. Run resolver dry-run.
11. Review assignment exceptions.
12. Run controlled apply.
13. Confirm Workbench data.
14. Confirm Manager Panel.
15. Confirm alert dry-run.
16. Move alerts to TestRecipientOnly.
17. Approve live alert mode.

Do not broaden resolver apply or alert sending until the branch readiness checklist passes.
