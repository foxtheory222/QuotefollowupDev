# Workbench Navigation

Workbench is the daily action center for quote follow-up, backorder follow-up, assignment issues, queue handoff, sticky notes, and manager/team review.

Browser validation confirmed the Workbench link appears in the Operations Hub branch rail and the Revenue Follow-Up Workbench Power App navigation. The Power App also shows Manager Panel and Admin Panel navigation in dev fallback mode because staff-to-systemuser mapping is not production-ready yet.

Power App navigation labels found:
- Workbench
- Manager Panel
- Admin Panel
- Staff
- Branch Memberships
- Staff Alias Mapping
- Branch Policies
- Assignment Exceptions
- Work Items
- Work Item Actions
- Alert Logs
