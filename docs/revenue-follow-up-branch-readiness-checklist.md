# Branch Readiness Checklist

Status: Ready for use.

For each branch, confirm:

- Branch exists and is active.
- Source feeds are configured.
- Active policy exists.
- Manager membership exists.
- GM membership exists.
- TSR memberships exist.
- CSSR memberships exist.
- AM Number aliases are mapped.
- CSSR Number aliases are mapped.
- Staff have verified primary emails.
- Staff are linked to Dataverse systemuser where needed.
- Resolver dry-run has no unexpected assignment exceptions.
- Controlled apply creates expected work items only.
- Workbench loads branch data.
- Manager Panel shows team data.
- Alert dispatcher dry-run logs expected candidates.
- No duplicate work item source keys.
- No duplicate assignment exception keys.
- No unapproved alerts are sent.
