# Manager Panel Visibility

Desired access:
- Manager
- GM
- Admin

Phase 7 result:
- Manager Panel exists and opens in the authenticated dev session.
- Team View/Manager mode remains available through the Workbench experience.
- No production Manager/GM memberships exist yet.
- Visibility is not production-enforced yet.

Blocked production behavior:
Production Manager Panel hiding requires verified staff-to-systemuser mapping plus Manager/GM/Admin memberships.

Friendly access-denied:
The desired friendly access-denied state is documented for Phase 8 implementation with final security roles. It was not fully enforceable in Phase 7 because production role context is incomplete.
