Browser validation confirmed these readiness views opened in the authenticated dev context.

## What Admins Use This For

- Fill staff primary email.
- Link staff to Dataverse systemuser.
- Review missing Manager/GM/Admin memberships.
- Review skipped alerts and missing recipients.
- Review assignment exceptions and overdue work items.
- Confirm policies before enabling test-recipient or live alert modes.
