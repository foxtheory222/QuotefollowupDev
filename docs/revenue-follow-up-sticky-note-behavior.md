# Sticky Note Behavior

Sticky notes are stored on `qfu_workitem.qfu_stickynote`.

## Phase 4B Behavior
- Sticky note is shown near the top of the detail panel.
- Sticky note can be edited inline.
- Saving patches `qfu_stickynote`.
- Saving updates `qfu_stickynoteupdatedon`.
- `qfu_stickynoteupdatedby` is not guessed because current user-to-staff mapping is not ready.
- Sticky note does not create an action row in Phase 4B.

## Preservation
Resolver/import behavior must not overwrite sticky note fields.
